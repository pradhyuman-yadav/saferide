Voice & tone

Calm, never cold
Reassuring like a trusted neighbour. Never sterile or corporate. Parents are anxious — we make them feel held.
Precise, never terse
Every word earns its place. Short sentences. No filler. But never so clipped it feels robotic.
Grounded, never showy
Premium through restraint. No exclamation marks. No "Amazing!". Let the product speak.
Human, never casual
Warm and personal but always professional. This is a child's safety. We take that weight seriously.

Do & don't

Do
Use Forest on Stone or Mist for all body text
Use DM Serif Display for emotional moments
Use gold sparingly — alerts, premium badges, warmth accents
Keep UI surfaces mostly Stone + White
Use Slate for metadata, timestamps, supporting info
Let whitespace breathe — never crowd elements
Don't
Never use Sage text on Mist background — low contrast
Never use gradients, shadows, or blur effects
Never use more than 3 palette colors in a single component
Never use ALL CAPS in body copy — subheadings only
Never use red — use Gold for alerts (red breaks the calm)
Never use font-weight 700+ — 500 is the heaviest
